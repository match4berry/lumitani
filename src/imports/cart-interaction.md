Tambahkan interaksi pada tombol keranjang (floating cart button) di bagian tengah bottom navigation pada aplikasi mobile marketplace pertanian "LumiTani".

Saat ini terdapat tombol keranjang berwarna hijau dengan ikon cart di tengah navigasi bawah. Buat fungsi ketika tombol ini ditekan.

1. Perilaku Tombol Keranjang

Ketika pengguna menekan tombol keranjang, arahkan pengguna ke halaman "Keranjang Belanja".

Halaman ini menampilkan semua produk yang telah ditambahkan oleh pengguna sebelum melakukan checkout.

2. Struktur Halaman Keranjang

Bagian atas:
Judul halaman: "Keranjang Belanja"

Daftar produk dalam keranjang:
Setiap item menampilkan:

* Gambar produk
* Nama produk
* Harga per kg
* Jumlah produk (dengan tombol + dan -)
* Subtotal harga
* Tombol hapus produk

3. Ringkasan Pesanan

Di bagian bawah halaman tampilkan ringkasan:

* Total harga
* Tombol utama: "Checkout"

Tombol checkout digunakan untuk melanjutkan ke proses pemesanan.

4. Kondisi Keranjang Kosong

Jika belum ada produk yang dimasukkan ke keranjang, tampilkan pesan:

"Keranjang kamu masih kosong"

Tambahkan tombol:
"Mulai Belanja"

Tombol ini mengarahkan pengguna kembali ke halaman Katalog Produk.

5. Desain UI

Gunakan desain yang konsisten dengan aplikasi LumiTani:

* warna utama hijau
* card produk dengan rounded corner
* layout mobile friendly
* tombol checkout berwarna hijau
* tampilan bersih seperti aplikasi marketplace grocery.

Tujuan fitur ini adalah memudahkan pengguna melihat produk yang ingin dibeli sebelum melakukan checkout.
